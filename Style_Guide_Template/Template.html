Paragraphs

Bulleted Lists

CSS selector:

ul, li

Sample code:

<ul> <li>This is a bullet item</li> </ul>

Rendered element:

  • This is a bullet item
  • This is a second bullet item

Numbered Lists

CSS selector:

ol, li

Sample code:

<ol> <li>This is a bullet item</li> </ol>

Rendered element:

  1. This is a bullet item
  2. This is a second bullet item
